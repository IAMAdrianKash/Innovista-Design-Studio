# Client Logos Directory

Add your client logo files here to display them in the scrolling logo strip on the homepage.

## File Naming Convention

Name your logo files as:
- `client-1.svg` (or .png)
- `client-2.svg` (or .png)
- `client-3.svg` (or .png)
- etc.

## Recommendations

- **Format**: SVG preferred for best quality and scalability
- **Size**: Logos will display at 24px-32px height
- **Color**: Logos will be displayed in grayscale with 25% opacity (full color on hover)
- **Background**: Use transparent backgrounds
- **Orientation**: Horizontal/landscape orientation works best

## Update the Component

If you need to add/remove logos, edit the `logos` array in:
`/components/LogoStrip.tsx`

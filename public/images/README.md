# Image Assets Directory

## Brand Assets (`/public/images/brand/`)

Upload your logo files here:

- **logo.svg** - Primary logo (SVG preferred for scalability)
- **logo.png** - PNG fallback (recommended: 512x512px transparent background)
- **logo-light.svg** - Light version for dark backgrounds (optional)
- **logo-dark.svg** - Dark version for light backgrounds (optional)
- **favicon.ico** - Browser favicon (32x32px or 16x16px)

## Open Graph Images (`/public/images/og/`)

Upload your social sharing images here:

- **og-default.jpg** - Default Open Graph image (1200x630px)
- **og-home.jpg** - Homepage specific OG image (1200x630px) - optional
- **og-case-studies.jpg** - Case studies page OG image (1200x630px) - optional
- **og-blog.jpg** - Blog page OG image (1200x630px) - optional

### Image Specifications

**Open Graph Images:**
- Recommended size: 1200x630px (1.91:1 aspect ratio)
- Format: JPG or PNG
- Max file size: 5MB
- Safe zone: Keep important content in center 1200x600px

**Logo:**
- SVG preferred for crisp scaling
- PNG should have transparent background
- Ensure logo works on both light and dark backgrounds
